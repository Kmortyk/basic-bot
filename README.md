# basic-bot

Простейший испольнитель команд на языке Basic.
Поддерживаемые конструкции: 
- up, down, left, right
- lookUp, lookDown, lookLeft, lookRight
- PRINT ...
- IF ... THEN ... ELSE ...
- FOR ... TO ... NEXT
- WHILE ... WEND
- GOTO label
- GOSUB label, RETURN
